# PC構成分析・推奨アプリ (PCA & Preference Analysis)

データサイエンスの手法（主成分分析）を用いて、PCのスペック構成を可視化し、ユーザーの用途によって最適なPCが異なることを示す目的で作成されたデスクトップアプリケーションです。

![Python](https://img.shields.io/badge/Python-3.10%2B-blue)
![PySide6](https://img.shields.io/badge/GUI-PySide6-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

## 🌟 主な機能

- **主成分分析 (PCA) による可視化**: 
  - 行中心化（Row-Centering）を用いたPCAにより、単なる「性能の高さ」ではなく「CPU寄りかGPU寄りか」といった**構成の偏り**を2次元空間に抽出します。
  - 点のサイズは「総合性能」、色は「価格」を表し、多角的な分析が可能です。
- **インタラクティブな推奨システム**:
  - スライダーで調整すると、リアルタイムで最適なPCがハイライトされます。
  - 予算フィルター機能により、現実的な選択肢に絞り込むことができます。
- **プリセット機能**:
  - 「プログラマー」「ゲーマー」「動画編集者」など、用途に合わせた最適な重み設定をワンクリックで適用できます。
- **CSVデータ管理**:
  - アプリ内でPCデータの追加・削除・編集・保存が完結します。

## 🚀 セットアップ

```bash

```

## 📊 分析の仕組み

1. **行中心化 PCA**:
   - 各PCの平均的な性能を差し引くことで、スケール（価格に比例しやすい総合性能）の影響を排除し、純粋な「スペックのバランス」を解析します。
2. **嗜好ベクトルによるスコアリング**:
   - ユーザーが設定した重みをPCA空間上のベクトルとして扱い、各PCとの内積（方向の一致度）を計算して適合スコアを算出します。
3. **XAI (説明可能なAI) 要素**:
   - 各スペックが主成分にどの程度寄与しているかを数値で表示し、分析結果の根拠を透明化しています。
